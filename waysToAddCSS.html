There are 3 ways

    External CSS: creating new file with .css extention and linking that file with html. The "link" element is used in an html document to tell the browser where to find the css file used to style the page.

    • The link use 3 attributes href : specifies the path, type : specifies the type of the document and the value should be text/css, rel : specifies the relationship between the html page and the file it is linked to.
    • We can use same link(stylesheet) in another html file. We can use more than one css stylesheet, then priority will be given to last property

    Internal CSS: we can include css rules within a html page by placing them inside a "style" element which usually sits inside the "head" element of the page. we can also add style element after closing "body" tag but not recommended. we can add type

    Inline CSS:style applied directly to an html element using the style attribute and style can be added to any element within the opening tag using style attribute. it's applicable for only to that element and we can add more than one property.
