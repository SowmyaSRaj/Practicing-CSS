What is CSS

  • CSS stands for cascading style sheets. CSS is also one of the core technologies of the web.
  • We use html to describe the content of a web page
  • We use css to describe the visula style of the content written in HTML.
  • It saves a lot of work. it can control the layout of multiple web pages all at once.
  • CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

What is CSS Rule

  • A css rule is a grouping of one or more css properties which are to be applied to one or more targetted html elements.
  • A css rule consists of a css selector and a declaration block
    • Selector - indicate which element the rule applies to. the same rule can be applied to more than one element if you separate the element names with commas.
    • Declaration block - it is a set of css properties, indicates how the elements referred to in the selector should be styled. declarations are separated into two parts a property and a value, and are separated by a colon. multiple css declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.
    • selector { property: value; }
